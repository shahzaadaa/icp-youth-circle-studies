Initial version of the ICP Youth Circle website.

This project is an all-in-one educational platform for students, covering school, college, and university levels.

Current features include:
- Homepage layout for the education hub
- Sections for MCQs practice, study notes, admissions information, and educational updates
- Basic HTML and CSS structure for future expansion

This is the foundation of the project and will be expanded with additional pages and features in upcoming updates.
